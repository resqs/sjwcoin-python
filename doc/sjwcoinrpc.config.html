sjwcoinrpc.config — Utilities for reading sjwcoin configuration files¶

Utilities for reading sjwcoin configuration files.

sjwcoinrpc.config.read_config_file(filename)¶

Read a simple '='-delimited config file. Raises IOError if unable to open file, or ValueError if an parse error occurs.

sjwcoinrpc.config.read_default_config(filename=None)¶

Read sjwcoin default configuration from the current user’s home directory.

Arguments:

  • filename: Path to a configuration file in a non-standard location (optional)
